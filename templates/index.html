Genrify

Hear it. Genrify it.




This small web app uses a neural network to predict the genre of a song you upload. The network can predict one of ten genres: pop, metal, disco, blues, reggae, classical, rock, hip-hop, country, and jazz.

To use it, select a music file from your computer to be fed into the genre classifier, and then click the Predict button below. This app currently supports .wav file type.
